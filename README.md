# ⚡ ResumeIQ — AI Resume Analyzer
 
Full-featured AI resume analysis powered by Groq (LLaMA 3.3 70B). 100% free, no billing required.
 
## Features
- 🎯 ATS Score with breakdown radar chart
- 📝 Cover Letter Generator (5 tone styles)
- 🎤 Interview Prep Kit (behavioral, technical, situational)
- ✏️ Bullet Point Rewriter
- ⚔️ Resume A/B Comparison
## Setup
 
1. Install dependencies:
   pip install -r requirements.txt
2. Get a free Groq API key at https://console.groq.com
3. Create a .env file:
   GROQ_API_KEY=your_key_here
4. Run:
   streamlit run app.py
